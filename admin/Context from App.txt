# 🔹 Key Logic & Data Flow

## Cart Flow

```js
Add → Redux → Update qty → Save to localStorage
Reload → loadCart() → hydrate state
```

---

## Order Flow

```js
Cart → format items → POST /orders → get orderId → redirect → clear cart
```

---

## Stats Calculation

```js
Loop orders → count items → count restaurants → sort → pick top
```

---

## Axios Setup

```js
api.interceptors.request.use(config => {
  const token = localStorage.getItem("token");
  if (token) config.headers.Authorization = `Bearer ${token}`;
});
```
